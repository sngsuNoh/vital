# Vital - 스킨케어 특허 검색 엔진

![Vital Logo](https://img.shields.io/badge/Vital-Patent%20Search-4285F4?style=for-the-badge)

대한민국 스킨케어 분야 특허 7,758건을 검색할 수 있는 웹 기반 검색 엔진입니다.

## 🌟 주요 기능

- **7,758개 특허 데이터베이스**: 대한민국 스킨케어 분야 특허 전체 검색
- **의미 기반 검색**: 키워드 매칭 기반 관련성 점수 계산
- **유사도 시각화**: 0-100% 범위의 유사도 점수 제공
- **직관적인 UI**: Google 스타일의 깔끔한 인터페이스
- **빠른 검색**: 클라이언트 사이드 검색으로 즉시 결과 제공

## 🔍 검색 사용법

### 메인 페이지
1. 검색창에 스킨케어 관련 키워드 입력 (예: "히알루론산 보습 화장료")
2. **Enter** 키 또는 **검색** 버튼 클릭
3. 검색 결과 페이지로 자동 이동

### 검색 결과 페이지
- 각 특허 문서에 대해 다음 정보 표시:
  - **제목**: 발명의 명칭
  - **출원번호**: 특허 출원번호
  - **요약**: 특허 요약 내용
  - **유사도**: 검색 쿼리와의 관련성 (0-100%)

### 검색 팁
- 구체적인 성분명 사용: "히알루론산", "레티놀", "나이아신아마이드"
- 기능성 키워드 활용: "보습", "미백", "주름 개선", "각질 제거"
- 제형 관련 키워드: "앰플", "세럼", "크림", "마스크"

## 📊 검색 알고리즘

### 점수 계산 방식 (다층적 분석)

1. **위치 기반 가중치**:
   - 제목 매칭: 빈도당 8점 (최대 25점)
   - 요약 매칭: 빈도당 4점 (최대 15점)
   - 본문 매칭: 빈도당 2점 (최대 12점)

2. **토큰 특성 고려**:
   - 4글자 이상 토큰: 10% 보너스
   - 6글자 이상 토큰: 추가 15% 보너스
   - 긴 토큰일수록 높은 가중치

3. **커버리지 보너스**:
   - 100% 매칭: 30% 보너스
   - 80% 이상 매칭: 15% 보너스
   - 60% 이상 매칭: 5% 보너스
   - 40% 미만 매칭: 30% 감점

4. **토큰 근접성 분석**:
   - 50자 이내: +5점
   - 100자 이내: +3점
   - 200자 이내: +2점
   - 500자 이내: +1점

5. **문서 길이 정규화**:
   - 200자 미만: 10% 감점
   - 5000자 이상: 5% 감점

6. **페널티 시스템**:
   - 단일 토큰 검색: 35% 감점

### 점수 표시
- **소수점 1자리** 표시 (예: 43.5%, 67.8%, 91.2%)
- 다양한 요소를 고려하여 매우 세밀한 점수 차이 반영
- 동일한 점수가 반복되는 문제 해결

### 100% 유사도 달성 조건
- 모든 쿼리 토큰이 제목과 본문에 여러 번 등장
- 토큰들이 문서 내에서 가까이 위치
- 긴 토큰 위주의 쿼리
- 2개 이상의 토큰으로 구성된 쿼리

### 최소 유사도 임계값
- 기본값: **20%**
- 임계값 미만인 경우: "유사한 특허를 발견할 수 없습니다" 메시지 표시
- 완벽한 매칭: 모든 쿼리 토큰이 문서와 제목에 정확히 일치해야 100% 달성

## ⚠️ 주의사항

1. **파일 크기**: `patents.json`이 11MB이므로 초기 로딩에 시간이 걸릴 수 있습니다
2. **브라우저 호환성**: 최신 Chrome, Firefox, Safari, Edge 권장
3. **검색 성능**: 클라이언트 사이드 검색이므로 대량 데이터 처리 시 성능 제한 있음
4. **CORS**: 로컬 테스트 시 HTTP 서버 필요 (file:// 프로토콜 사용 불가)
